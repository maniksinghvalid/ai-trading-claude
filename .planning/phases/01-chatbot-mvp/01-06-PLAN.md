<objective>
Slice 5: Build the Next.js 14 App Router streaming chat UI — an async-generator client that reads
the backend SSE stream, a `ChatWindow` that renders tokens live, and a `MessageBubble` that shows a
Sources list from citations and continues a session across messages.

Purpose: Deliver the user-facing end-to-end experience — the phase's headline success criterion.
Output: A working browser chat that streams token-by-token with rendered sources and session
continuity.
</objective>

## Artifacts this phase produces (this plan)

- Next.js 14 App Router project (TypeScript + Tailwind, `--no-src-dir`) under `frontend/`
- `frontend/lib/types.ts` -> `ChatRequest`, `Citation`, `ChatResponse` mirrors of the backend Pydantic schemas
- `frontend/lib/api.ts` -> async generator `streamChat(message, sessionId?, ticker?)` POSTing to backend `/chat/stream`, splitting on `\n\n`, yielding `{event, data}`
- `frontend/.env.local.example` -> `NEXT_PUBLIC_API_BASE=http://localhost:8000` (backend base URL the frontend reads)
- `frontend/components/ChatWindow.tsx` -> "use client" component (state: messages, input, sessionId, streaming; `send()` handling session/citations/token/done)
- `frontend/components/MessageBubble.tsx` -> ReactMarkdown render + "Sources" list when citations present
- `frontend/app/page.tsx` -> server component rendering `<ChatWindow/>`; plus layout/globals/config

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Tampering | XSS via markdown render | mitigate | Render through react-markdown (no raw HTML / no dangerouslySetInnerHTML; do not enable rehype-raw) |
| T-06-02 | Information Disclosure | CORS exposure | mitigate | Backend CORS limited to settings.cors_origins (http://localhost:3000) |
| T-06-03 | Spoofing | citations point to non-existent sources | mitigate | Sources list renders only from citations emitted by the backend (real chunk metadata) |
| T-06-SC | Tampering | npm install ai/react-markdown/eventsource-parser | mitigate | Mainstream packages from CONTEXT.md locked list; verify on npmjs.com before install |
