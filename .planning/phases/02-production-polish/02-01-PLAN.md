<objective>
Add automatic ticker extraction + intent classification so the chatbot no longer requires the
caller to pass `ticker` explicitly (TICK-01), and author the cross-cutting schema-regression
test (VERIFY-SCHEMA) that asserts the upstream Pinecone metadata contract. This is slice 6 of
`plan/trading-chatbot.md`.

Purpose: Phase 1 relied on the frontend's optional `ticker` input. Phase 2 resolves the ticker
and intent from the natural-language message itself, with coreference inheritance from session
history. The intent label is also the signal slice 7 uses to decide when to fetch a live quote.

Output: `ticker_extractor.py`, `intent_classifier.py`, a modified `/chat` + `/chat/stream` flow
that resolves ticker+intent before retrieval, and the VERIFY-SCHEMA regression test.
</objective>
