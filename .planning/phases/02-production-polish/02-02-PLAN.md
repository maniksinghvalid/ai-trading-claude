<objective>
Add a live market-data quote layer (slice 7, QUOTE-01): a `market_data.py` yfinance wrapper with
a ~15-minute in-memory cache, a `GET /quote/{ticker}` endpoint, and `/chat` logic that fetches a
live quote ONLY when the slice-6 intent classifier signals a current-price question — rendering
it distinctly from cited memory chunks.

Purpose: Outlook/thesis questions are answered from cited memory; "what's it trading at?"
questions need fresh price data. The intent label from 02-01 is the routing signal. yfinance is
~15 min delayed, so every quote carries a timestamp and a source label.

Output: `market_data.py`, `routes/quote.py`, intent-gated quote injection in `/chat` +
`/chat/stream`, and the `prompts.py` live-quote inset wired through the existing `live_quote` arg.
</objective>
